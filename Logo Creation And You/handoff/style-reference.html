∗How to hand this to Claude Code

The handoff package

  1. Copy the entire handoff/ folder from this project into the root of your &You repo.
  2. In your Claude Code session, prompt:
    Read handoff/CLAUDE.md and apply the redesign. Don't change app logic, only visuals.
  3. Claude Code will read the style guide, merge the Tailwind config, drop the tokens into your global CSS, and rewrite components to match the recipes.
  4. Replace /public/apple-touch-icon.png & favicon with the ones in handoff/icons/.
Tip: keep this style-reference.html open in a browser tab while iterating. It's the visual ground truth.
